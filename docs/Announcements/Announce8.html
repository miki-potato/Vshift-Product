🎉 HTML対応お知らせ機能がリリースされました！

V-Shiftアプリのお知らせ機能が大幅にアップデートされました。これまでテキストのみだったお知らせが、リッチなHTMLコンテンツで表示できるようになりました。

✨ 新機能のポイント：
CSSスタイル、画像、リンクなど、Webページと同様の表現力でお知らせを作成できます。

🚀 主な機能

  • 🎨 CSSスタイルの完全サポート
  • 🖼️ 画像の埋め込み表示
  • 🔗 リンクの設置
  • 📱 レスポンシブデザイン対応
  • ⚡ 高速レンダリング（WebView2使用）

📝 使用方法

お知らせは以下のURLで配信されます：

https://miki-potato.github.io/Vshift-Product/Announcements/Announce0.html

HTMLファイルを作成してアップロードするだけで、美しいお知らせが表示されます。

🔄 レガシー対応

既存のJSON形式のお知らせも引き続きサポートされています。段階的にHTML形式に移行できます。

このお知らせは新しいHTML機能で作成されました 🎊
